# Reference: PCI DSS v4.0

## Key Requirements

- **Req 3**: Protect Stored Account Data.
- **Req 4**: Protect Cardholder Data with Strong Cryptography During Transmission.
- **Req 10**: Log and Monitor All Access.
- **Req 11**: Test Security of Systems and Networks Regularly.

## SAQ A vs SAQ D

- **SAQ A**: Card-not-present, all processing outsourced (i.e. using iframes/redirects).
- **SAQ D**: Service Provider eligible.
