# Reference: Input Attributes

## Input Modes

- `text`: Standard.
- `decimal`: Numbers + Decimal point.
- `numeric`: Numbers (0-9). Best for PINs/CC.
- `tel`: Phone pad (includes \* and #). Good for older devices but semantically wrong for CC.
- `email`: Includes @.
- `url`: Includes .com / /.

## Autocomplete Values

- `cc-number`
- `cc-exp`
- `cc-csc`
- `one-time-code` (Crucial for 2FA)
