# Reference: Mobile Payment Page Design

## Viewport

- `<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1">` (Prevent zoom on input focus for iOS).

## Touch Targets

- Buttons: Min `44px` height (Apple Human Interface Guidelines).
- Inputs: `16px` font size min to avoid auto-zoom.

## Sticky Elements

- On mobile, keeping the "Pay $100" button sticky at bottom is high conversion but ensure it doesn't cover input fields (keyboard awareness).
