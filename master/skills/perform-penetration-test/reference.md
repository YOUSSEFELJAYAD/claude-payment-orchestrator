# Reference: OWASP Top 10

## Key Vectors

- **A01: Broken Access Control**: Can User A see User B's transaction?
- **A03: Injection**: SQLi (Test `' OR 1=1`), Command Injection.
- **A07: Identification Failures**: weak passwords, session fixation.

## Tools

- **ZAP (Zed Attack Proxy)**: Open Source.
- **Burp Suite**: Industry Standard (Paid).
- **Kali Linux**: Distro with pre-installed tools.
