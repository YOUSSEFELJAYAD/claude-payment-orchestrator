# Reference: OpenAPI 3.0

## Components

- **Paths**: The endpoints (`/payments`).
- **Schemas**: Data models (`PaymentRequest`).
- **SecuritySchemes**: Auth methods (`BearerAuth`).

## Zod to OpenAPI

Use `zod-to-openapi` or `next-swagger-doc` to auto-generate docs from backend validation schemas. This ensures code and docs never drift.
