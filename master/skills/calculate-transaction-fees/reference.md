# Reference: Fee Models

## Blended (Flat Rate)

- **Structure**: `2.9% + $0.30`
- **Pros**: Predictable. Simple reconciliation.
- **Cons**: Expensive for Debit/Low-risk cards. Hidden margins.

## Interchange++ (IC++)

- **Structure**: `IC Fee + Scheme Fee + Acquirer Markup`
- **Example**: `0.2% (IC) + 0.05% (Scheme) + 0.5% (Markup)` = `0.75%` Total.
- **Pros**: Transparent. Cheaper for large volume/domestic.
- **Cons**: Variable monthly costs.

## Key Definitions

- **Interchange**: Paid to Issuer (Customer's Bank).
- **Scheme**: Paid to Network (Visa/MC).
- **Markup**: Paid to Acquirer/Platform.
